window객체

-자바 스크립트의 최상위 객체이다.
-Dom(Document Object Model =문서 객체 모델), BOM(Brower Object Model = 브라우저 객체 모델),javascript내장 객체 등을 가지고 있다.

DOM(Document Object Model =문서 객체 모델)

-노드 트리 구조의 웹문서 관련 태그(element), 텍스트(text),속성(attribute)등의 내장객체 및 함수를 가지고있다.
-전반적으로 html의 구조, 스타일 , 이벤트에 관련된 기능을 설정할 때 용이한 객체이다.
-예시 : document,style , forms , body , innerText ,innerHTML 등의 내장객체와 해당 객체에서 제공되는 함수들이 존재함.

BOM(Brower Object Model = 브라우저 객체 모델)

- 자바스크립트와 브라우저간의 소통을 위해 만들어진 모델이다. - 전반적으로 브라우저와 그 이상의 외부환경을 설정할 때 용이한 객체이다.
- BOM의 대표적인 내장객체 : screen
- 예시 : screen,navigaator, location, history 등의 내장객체와 해당 객체에서 제공되는 함수들이 존재함.

javascript내장 객체

- 자바스크립트에서 제공되는 내장객체들을 말한다.
- 함수형 내장객체(표준내장객체)와 표준내장객체에서 제공되는 객체들, 함수들이 존재한다.
- 예시 : Array , Number , Function , arguments 등의 내장객체와 해당 객체에서 제공되는 함수들이 존재함.
