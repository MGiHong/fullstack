the vertical-align property

vertical-align은 오직 inline과 table-cell 엘리먼트에만 적용된다 block level 엘리먼트에는 사용할 수 없다.

the vertical-align :baseline(default)

해당 요소의 하단을 부모 요소의 baseline을 기준으로 맞춰 정렬한다.
baseline 이없으면 부모 요소의 아래쪽 가장 자리를 기준으로 맞춰 정렬한다.

imge with a default alignment.

the vertical-align :top

해당 요소의 하단을 부모 요소의 위쪽 가장자리에 맞춰 정렬한다.

imge with a default alignment.

the vertical-align :middle

부모 요소의 x-height의 중간점을 기준으로 수직 가운데 정렬을 한다. x-height = 소문자 x의 높이 =1ex

imge with a default alignment.

the vertical-align :bottom

해당 요소의 하단을 부모 요소의 아래쪽 가장자리에 맞춰 정렬한다.

imge with a default alignment.
